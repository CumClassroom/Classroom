How to download the games:

STEPS

1. Go to the game you want to download and click on it.
2. It should look like this:
   <img width="1555" height="438" alt="image" src="https://github.com/user-attachments/assets/4af04717-8cbe-43ff-9e61-39e5ef234c5e" />
3. In the top right click copy:
<img width="1370" height="390" alt="image" src="https://github.com/user-attachments/assets/fb771c08-404f-4397-aa4e-cbf1908d08a3" />
4. Open up notepad on your computer paste the code in and select file then save as:
<img width="622" height="421" alt="image" src="https://github.com/user-attachments/assets/8b771bd4-00d2-4473-967c-994cf4318a7b" />
5. Change the save as type to "All Files":
<img width="451" height="93" alt="image" src="https://github.com/user-attachments/assets/1a383fb8-aa43-41a4-a286-5b76d5778de2" />
6. Name the file to whatever the name is in github, so if the name is index.html in github; name it to that. (make sure to save it to somewhere recognizable like your desktop):
7. <img width="338" height="164" alt="image" src="https://github.com/user-attachments/assets/65efe08a-c629-4774-a47a-e094e8636e18" />
8. Run the file and make sure it opens up in either your edge or chrome browser, Both should work.
9. <img width="482" height="442" alt="image" src="https://github.com/user-attachments/assets/f217d59e-418d-4b03-aa5c-3daec09fb2b5" />
